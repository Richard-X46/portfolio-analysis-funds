## Overview

This project analyzes Canadian mutual funds using OLS regression to predict future prices and applies Monte Carlo simulation for portfolio optimization.

- **Fund analysis.ipynb**: Main analysis and modeling notebook.
- **data/**: Contains source datasets for mutual funds and their price history.
- **notes.md**: Project notes, scope, and objectives.
- **proposal.md**: Project proposal and business context.

## How to Use

1. Review and run `Fund analysis.ipynb` for analysis and modeling.
2. Data files are located in the `data/` directory.
3. See `proposal.md` for project details.
